# Portfolio — Fidel Cedric Odoyo

Personal portfolio site built with Next.js 14, Tailwind CSS, and Framer Motion. Deployed on GitHub Pages.

**Tech Stack:** Next.js · TypeScript · Tailwind CSS · Framer Motion · lucide-react

**Live:** https://Polymerthcedric.github.io/portfolio

## Development

```bash
npm install
npm run dev      # → http://localhost:3000
npm run build    # static export to /out
```

## Deploy

Push to `main` → GitHub Actions builds and deploys to GitHub Pages automatically.
